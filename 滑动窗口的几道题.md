# 滑动窗口的几道题
## 1.知识点：
滑动窗口本质上是一个双指针问题，两个指针分别代表窗口的左右边界，窗口内满足某些条件时，右指针向右移动扩大窗口，否则左指针向右移动缩小窗口。
### 大体框架：
```cpp
    // 索引区间 [left, right) 是窗口
int left = 0, right = 0;//窗口的左右边界

while (right < nums.size()) {
    // 增大窗口
    window.addLast(nums[right]);
    right++;
    
    while (window needs shrink) {
        // 缩小窗口
        window.removeFirst(nums[left]);
        left++;
    }
}
```
## 题目1：最小覆盖子串
给你一个字符串 s 、一个字符串 t 。返回 s 中涵盖 t 所有字符的最小子串。如果 s 中不存在涵盖 t 所有字符的子串，则返回空字符串 "" 。
注意：如果 s 中存在这样的子串，我们保证它是唯一的答案。对于 t 中重复出现的字符，我们寻找的子串中该字符出现的次数必须不少于 t 中该字符出现的次数。
### 思路：
1. 我们设定窗口为[left,right)，初始时left=0，right=0。我们先不断增大窗口，直到窗口内的字符串包含t中的所有字符。
2. 当窗口内的字符串包含t中的所有字符时，我们记录下当前窗口的长度，并尝试缩小窗口，即移动left指针，直到窗口内的字符串不再包含t中的所有字符。
3. 我们设定need数组记录t中每个字符的出现次数，window数组记录当前窗口内每个字符的出现次数。当window满足need时，我们就尝试缩小窗口。need和window的比较可以通过一个变量valid来记录满足条件的字符个数，当valid等于need中不同字符的个数时，说明窗口内包含t中的所有字符。
### 代码实现：
```cpp
class Solution{
    public:
        string minWindow(string s, string t){
            //用哈希表记录t中每个字符的出现次数和窗口内每个字符的出现次数
            unordered_map<char ,int> need,window;//need记录t中每个字符的出现次数，window记录当前窗口内每个字符的出现次数
            for(char c:t){
                need[c]++;//统计t中每个字符的出现次数
            }
            int left=0,right=0;
            int valid=0;//记录满足条件的字符个数
            int start=0,len=INT_MAX;//记录最小覆盖子串的起始位置和长度
            //这一部分是增大窗口，当窗口内的字符串包含t中的所有字符时，尝试缩小窗口
            while(right<s.size()){
                char c=s[right];//c是将要移入窗口的字符
                right++;//右移窗口
                if(need.count(c)){
                    window[c]++;//更新窗口内该字符的出现次数
                    if(window[c]==need[c]){//如果窗口内该字符的出现次数等于need中该字符的出现次数，说明满足条件
                        valid++;//满足条件的字符个数加1
                    }
                }
                //这一部分是缩小窗口，当窗口内的字符串包含t中的所有字符时，尝试缩小窗口
                while(valid==need.size()){
                    if(right-left<len){
                        start=left;//更新最小覆盖子串的起始位置
                        len=right-left;//更新最小覆盖子串的长度
                    }
                        char d=s[left];//d是将要移出窗口的字符
                        left++;
                        if(need.count(d)){
                            if(window[d]==need[d]){
                                valid--;//满足条件的字符个数减1
                            }
                            window[d]--;//更新窗口内该字符的出现次数,这个必须在外面，因为无论满足条件与否都要更新窗口内该字符的出现次数
                        }
                    }
                }
            
            return len==INT_MAX?"":s.substr(start, len);
        }
};
```
上面valid==need.size()的判断条件是当窗口内满足条件的字符个数等于need中不同字符的个数时，说明窗口内包含t中的所有字符，此时我们就尝试缩小窗口。直到窗口内不再满足条件，即valid<need.size()，此时我们就停止缩小窗口，继续增大窗口。
## 题目2：字符串的排列
给你两个字符串 s1 和 s2 ，写一个函数来判断 s2 是否包含 s1 的排列。如果是，返回 true ；否则，返回 false 。换句话说，s1 的排列之一是 s2 的 子串 。
### 思路:
和上面的思路基本一致，主要区别是移动左边窗口的时间，也就是right-left>=s1.size()时才移动左边窗口。另外，只要出现valil==need.size()就说明窗口内包含s1的排列了。
### 代码实现：
```cpp
class Solution{
    public:
        bool checkInclusion(string s1,string s2){
            unordered_map<char,int>need,window;
            for(char c:s1){
                need[c]++;
            }
            int left=0,right=0;
            int valid=0;
            int start=0,len=INT_MAX;
            while(right<s2.size()){
                char c=s2[right];
                right++;
                if(need.cont(c)){
                    window[c]++;
                    if(window[c]==need[c]){
                        valid++;
                    }

                }
            }
            while(right-left>s1.size()){
                if(valid==need.size()){
                    return true;
                
                char d=s2[left];
                left++;
                if(need.count(d)){
                    if(window[d]==need[d]){
                        valid--;
                    }
                    window[d]--;
                }
            }
        }
        return false;
}
};
```
## 题目3：找出字符串中所有字母异位词
给定一个字符串 s 和一个非空字符串 p，找到 s 中所有是 p 的字母异位词的子串，返回这些子串的起始索引。字符串只包含小写英文字母，并且字符串 s 和 p 的长度都不超过 20100。异位词指字母相同，但排列不同的字符串。
### 思路：
本质上就是找出s中所有包含p中所有字符的子串，和上面的题目类似，只不过需要记录下所有满足条件的子串的起始索引。
### 代码实现：
```cpp
#include <vector>
#include <string>
#include <unordered_map>

using namespace std;

class Solution {
public:
    vector<int> findAnagrams(string s, string t) {
        unordered_map<char, int> need;
        unordered_map<char, int> window;
        for (char c : t) {
            need[c]++;
        }

        int left = 0, right = 0;
        int valid = 0;
        // 记录结果
        vector<int> res;
        while (right < s.size()) {
            char c = s[right];
            right++;
            // 进行窗口内数据的一系列更新
            if (need.count(c)) {
                window[c]++;
                if (window[c] == need[c]) {
                    valid++;
                }
            }
            // 判断左侧窗口是否要收缩
            while (right - left >= t.size()) {
                // 当窗口符合条件时，把起始索引加入 res
                if (valid == need.size()) {
                    res.push_back(left);
                }
                char d = s[left];
                left++;
                // 进行窗口内数据的一系列更新
                if (need.count(d)) {
                    if (window[d] == need[d]) {
                        valid--;
                    }
                    window[d]--;
                }
            }
        }
        return res;
    }
};
```
## 题目4：最长无重复子串
给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。
### 思路：
这道题连need和valid都不需要了，只需要一个window数组记录当前窗口内每个字符的出现次数，当窗口内某个字符的出现次数大于1时，说明窗口内有重复字符了，此时就需要缩小窗口。还有就是注意在哪里更新result，应该在缩小窗口之前更新，因为缩小窗口之后窗口内就不满足条件了。
### 代码实现：
```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char, int> window;

        int left = 0, right = 0;
        // 记录结果
        int res = 0;
        while (right < s.size()) {
            char c = s[right];
            right++;
            // 进行窗口内数据的一系列更新
            window[c]++;
            // 判断左侧窗口是否要收缩
            while (window[c] > 1) {
                char d = s[left];
                left++;
                // 进行窗口内数据的一系列更新
                window[d]--;
            }
            // 在这里更新答案
            res = max(res, right - left);
        }
        return res;
    }
};
```
