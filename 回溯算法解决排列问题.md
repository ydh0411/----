# 回溯算法
1. 对于排列组合和子集问题，本质上还是从nums按照某种规则取某些元素。比如：元素无重不可复选；元素可重不可复选；元素无重可复选
本质上还是穷举所有解，而且这些解呈现树形。需要记住组合树和排列树
2. 回溯算法的核心是：在树形结构中进行深度优先搜索，遇到不满足条件的解就回退（backtrack）到上一个状态，继续探索其他可能的解。
3. 组合树：每个节点代表一个元素，分叉代表是否选择该元素。比如：对于[1,2,3]，组合树如下：
一个思想是，我们通过保持元素之间的相对顺序不变来避免重复的组合。我们把根节点视为第0层，那么将每个节点和根节点之间树枝上的元素作为该节点的值，那么第 n 层的所有节点就是大小为 n 的所有子集。
```
          []
         / | \
        1  2  3
       /   \   \
      2     3   1
     /       \   \
    3         1   2
```
## 例题1：子集
1. 给定一个整数数组 nums ，数组中的元素 互不相同 。返回该数组所有可能的子集（幂集）。 解集 不能 包含重复的子集。你可以按 任意顺序 返回解集。
```输入：nums = [1,2,3]
输出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
```
2. 思路：回溯算法，组合树.子集是组合问题
3. 代码：
```cpp
class Solution{
    private:
    vector<vector<int>>res;
    vector<int>track;
    public:
    vector<vector<int>> subsets(vector<int>&nums){
        backtrack(nums,0);// 从第0个元素开始回溯
        return res;
    }
    void backtrack(vector<int>&nums,int start){
        res.push_back(track);//
        for(int i=start;i<nums.size();i++){//这里的start是为了保证元素之间的相对顺序不变，避免重复的组合
            track.push_back(nums[i]);// 选择当前元素，加入到track中
            backtrack(nums,i+1);// 递归调用，继续探索下一个元素
            track.pop_back();// 回退到上一个状态，继续探索其他可能的解
        }
    }
};
```
## 例题2：组合
1. 给定两个整数 n 和 k，返回 1 ... n 中所有可能的 k 个数的组合。
```输入：n = 4, k = 2
输出：[[2,4],[3,4],[2,3],[1,2],[1,3],[1,4]]
```
2. 思路：本质上和子集问题也一样，就是把第k层的所有节点作为解。回溯算法，组合树
3. 代码：
```cpp
class Solution{
    private:
    vector<vector<int>>res;
    vector<int>track;
    public:
    vector<vector<int>> combine(int n,int k){
        backtrack(n,k,1);// 从第1个元素开始回溯
        return res;
    }
    void backtrack(int n,int k,int start){
        //和上面的第一个代码其实就是这里的区别，其他的都是一样的
        if(track.size()==k){//
            res.push_back(track);// 当track的大小等于k时，说明找到了一个解，加入到res中
            return;// 回退到上一个状态，继续探索其他可能的解
        }
        for(int i=start;i<=n;i++){
            track.push_back(i);
            backtrack(n,k,i+1);// 递归调用，继续探索下一个元素
            track.pop_back();// 回退到上一个状态，继续探索其他可能的解
        }
    }
}
